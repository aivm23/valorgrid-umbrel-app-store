# ValorGrid Community App Store

Community App Store for installing ValorGrid on Umbrel during publication
validation.

Add this repository as a Community App Store in Umbrel:

```text
https://github.com/aivm23/valorgrid-umbrel-app-store
```

The app package is generated from the ValorGrid repository under
`deploy/umbrel/community-store` and pins the published GHCR image by version and
digest.
